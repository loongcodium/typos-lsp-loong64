# typos-lsp-loong64

This project builds [typos-lsp](https://github.com/tekumara/typos-lsp) for LoongArch with bundled binaries.

To get the extension, search for it in the extensions tab of VSCodium and install it directly.

To install the `.vsix` file offline, please get it from [releases](https://github.com/loongcodium/typos-lsp-loong64/releases), and then install it with `codium --install-extension`. Binary files are also provided.
